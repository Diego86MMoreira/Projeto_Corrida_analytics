# Projeto Corrida Analytics

Projeto de análise de treinos de corrida baseado em:

- Treinos planejados (PDF Sisrun)
- Treinos realizados (arquivos FIT)

## Objetivo

Transformar dados de corrida em insights de performance.

## Estrutura

- data/ → dados brutos e processados
- src/ → código do projeto
- docs/ → documentação
- notebooks/ → análises exploratórias
