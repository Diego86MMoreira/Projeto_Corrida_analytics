# Arquitetura do Projeto Corrida Analytics

## Objetivo
Analisar treinos de corrida e gerar insights de performance.

## Fontes de dados
- PDFs Sisrun (treinos planejados)
- Arquivos FIT (treinos realizados)

## Pipeline

1. Ingestão (raw)
2. Processamento (ETL)
3. Modelagem
4. Análise
5. Visualização
